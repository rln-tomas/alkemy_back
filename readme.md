# Alkemy challenge JS - Backend 

## Install: 
First clone the project: 
```bash
    git clone https://github.com/rln-tomas/alkemy_back
```

Install the dependencies: 
```bash
    cd alkemy_back && npm install
```
Create environment variables in .env file: 
Example: 
```bash 
    mv .env.example .env
```

## Running: 

```bash
    npm run start
```
